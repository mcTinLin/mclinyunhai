我们的项目



创神大陆

创神大陆是凌云海工作室精心发起的网易我的世界服务器项目。目前由实力出众的星月制作组负责制作，他们凭借丰富的经验和创新的思维，为服务器注入了全新的活力。在创神大陆中，玩家将置身于一个充满奇幻色彩的世界，这里有着广袤无垠的地图等待探索，还有各种独特的生物和神秘的遗迹。游戏内设有丰富多样的玩法，比如特色的神装系统，玩家可以通过不断冒险和挑战来收集材料，打造属于自己的强大神装，提升战斗能力。此外，神秘的副本探险玩法更是充满了刺激与惊喜，每个副本都有独特的机制和强大的 BOSS 等待玩家去征服。无论是喜欢独自探索的独行侠，还是热衷于团队合作的玩家，都能在创神大陆中找到属于自己的乐趣。

星梦起源

星梦起源是凌云海工作室倾力打造的 Minecraft Java 版服务器项目，由经验丰富且热情满满的天灵担任服主。该服务器致力于为玩家营造一个活跃、和谐且充满乐趣的社区氛围。在星梦起源中，玩家可以体验到多种精心设计的游戏模式。生存模式为玩家提供了一个原汁原味的Minecraft生存体验，玩家需要在这个世界中收集资源、建造家园、抵御怪物的袭击，感受从零开始建设的成就感。而竞技模式则充满了挑战与激情，玩家可以与其他高手一决高下，展示自己的操作技巧和策略思维。除此之外，服务器还会定期举办各种丰富多彩的活动，比如限时的打怪活动、建筑比赛等。这些活动不仅为玩家带来了丰厚的福利和奖励，还增进了玩家之间的交流与互动，让大家在游戏中结交到更多志同道合的朋友。
